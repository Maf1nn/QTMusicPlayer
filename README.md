***Qt Music Player***
Welcome! This is my Qt C++ audio music player project. I am building it into a full-featured desktop application, implementing my own ideas and exploring the capabilities of Qt.
If you are interested in following the development of this project, feel free to reach out to me on Telegram: @M1DA2

Current State
At this stage I have implemented the core logic and basic interface of a music player, including:

Track list with add/remove support
Play / Pause
Next / Previous track
Volume control
Save and load playlists (.m3u)


Planned Features

Account system and authorization
Download tracks from SoundCloud


Feel free to share your ideas and suggestions on Telegram!
